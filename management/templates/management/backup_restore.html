Important Note for Database Restore!

For data safety, database restore must be done from the command line when the server is stopped. Please follow these steps:

  1. Stop the server: Press Ctrl + C in the terminal where the server is running.
  2. Run restore command: Open a new terminal in your project folder and run:
    python manage.py dbrestore --uncompress -i YOUR_DATABASE_BACKUP_FILE.gz
  3. Restart the server: Run python manage.py runserver to start the application again.

Replace YOUR_DATABASE_BACKUP_FILE.gz with the actual filename from the list below.
